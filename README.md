# Cyrusdev - Modern Web Development

A professional portfolio website built with Astro, Tailwind CSS, and Node.js.

## 🚀 Tech Stack

- **Framework**: [Astro 5](https://astro.build)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com)
- **Runtime**: Node.js (Standalone adapter)
- **Languages**: TypeScript, JavaScript

## ✨ Features

- **Responsive Design**: Mobile-first approach using Tailwind CSS.
- **Server-Side Rendering (SSR)**: Dynamic content delivery with `@astrojs/node`.
- **Project Showcase**: Filtered and tagged project portfolio.
- **Contact Form**: Integrated with Cloudflare Turnstile for spam protection.

## 🛠️ Getting Started

1.  **Install dependencies**:
    ```sh
    pnpm install
    ```

2.  **Start development server**:
    ```sh
    pnpm run dev
    ```

3.  **Build for production**:
    ```sh
    pnpm run build
    ```

4.  **Preview production build**:
    ```sh
    pnpm run preview
    ```
